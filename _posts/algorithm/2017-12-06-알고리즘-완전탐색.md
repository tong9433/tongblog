# 완전탐색
* 가능한 모든 경우의 수를 만들어보고 탐색하는 방법

<br>
## 1. 그냥다해보기
* 그냥 다해보는 방법임.

ex) Baekjoon2309 일곱난쟁이
```java
import java.util.Arrays;
import java.util.Scanner;
public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int[] arr = new int[9];
        int sum = 0;
        for(int i=0;i<9;i++){
            arr[i] = sc.nextInt();
            sum += arr[i];
        }
        boolean flag = false;
        for(int i=0; i<9; i++){
            for(int j=i+1; j<9; j++){
                if(sum-arr[i]-arr[j]==100){
                   flag = true;
                   arr[i]=100000;
                   arr[j]=100000;
                   break;
                }
            }
            if(flag) break;
        }
        Arrays.sort(arr);
        for(int k=0; k<7;k++){
            System.out.println(arr[k]);
        }
    }
}
```

## 3. 순열 사용하기
* 문제의 조건이 N개를 다 하고 순서를 바꿀경우 사용할 수 있다.
* 단 모든 순열을 구하는 것은 총 개수 `N!`에 다음순열을 구하는 시간복잡도 `O(N)`
총 `O(N!*N)`의 시간복잡도가 소모된다.
* `10!*10 = 3천만` 정도 되므로 N이 10 이하일 때만 사용 가능.
* ex) Baekjoon10971 외판원 순회2
```java
import java.util.Scanner;

public class Baekjoon10971_외판원순회2_순열 {

    public static boolean next_permutation(int[] a) {
        int i = a.length-1;
        while (i > 0 && a[i-1] >= a[i]) {
            i -= 1;
        }

        //마지막 순열
        if (i <= 0) {
            return false;
        }

        int j = a.length-1;
        while (a[j] <= a[i-1]) {
            j -= 1;
        }

        int temp = a[i-1];
        a[i-1] = a[j];
        a[j] = temp;

        j = a.length-1;
        while (i < j) {
            temp = a[i];
            a[i] = a[j];
            a[j] = temp;
            i += 1;
            j -= 1;
        }
        return true;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[][] map = new int[n][n];

        for(int i=0; i<n; i++){
            for(int j=0; j<n; j++){
                map[i][j] = sc.nextInt();
            }
        }

        int[] arr = new int[n];

        int ans = 0;

        for(int i=0;i<n;i++){
            arr[i] = i;
            if(i!=n-1)ans+= map[i][i+1];
            if(i==n-1)ans+= map[n-1][0];
        }

        while(next_permutation(arr)){
            int sum=0;
            for(int i=0;i<n-1;i++){
                sum += map[arr[i]][arr[i+1]];
            }
            sum += map[arr[n-1]][arr[0]];

            if(ans>sum) ans =sum;
        }

        System.out.println(ans);
    }

}
```

* 배열이 11222 라는 수열로 되있어도 `next_permutation`이 수행됨.

## 4. 큐 사용하기 (BFS처럼)
* 최솟값을 수하는 문제에서 , 상태의 개수가 작고 가중치가 1일 경우 BFS이다.
* ex) Baekjoon1697 숨바꼭질

```java
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class Baekjoon1697_숨바꼭질_큐_완전탐색 {

    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int result = sc.nextInt();
        boolean[] check = new boolean[100001];
        int[] dist = new int[100001];

        Queue<Integer> q = new LinkedList<>();

        q.add(n);
        check[n] = true;
        dist[n] = 0;


        while(!q.isEmpty()){
            int x = q.remove();

            int[] nx = new int[3];
            nx[0] = 2*x;
            nx[1] = x-1;
            nx[2] = x+1;

            for(int k=0; k<3; k++){
                if(0<=nx[k]&&nx[k]<=100000){
                    if(check[nx[k]]==false){
                        q.add(nx[k]);
                        check[nx[k]]= true;
                        dist[nx[k]]=dist[x]+1;
                        if(nx[k]==result){
                            System.out.println(dist[nx[k]]);
                            System.exit(0);
                        }
                    }
                }
            }
        }
    }
}
```
